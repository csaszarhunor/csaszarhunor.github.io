THE PROJECT

This website is the result of a school project. The aim of the project is learning the basics of HTML, CSS and JS with the method of project based learning.

The tools I've used during this week are following:

  • Codecademy - for learning the syntax of HTML and CSS
  • W3.org Validator - to make sure the website is according to W3C standards
  • GitHub Pages - to host the website and for version controlling
  • Webstorm - to create the website
  • Google Developers - a good guideline for responsive websites
